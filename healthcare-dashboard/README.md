# Healthcare Dashboard Analysis

## Objective
Analyze healthcare data to identify trends in patient distribution, resource usage, and regional healthcare demand using Excel and Power BI.

## Tools Used
- Excel
- Power BI

## Dataset
The dataset includes:
- Patient information
- Regional distribution
- Healthcare service usage

## Project Workflow
1. Cleaned and prepared the dataset in Excel
2. Performed exploratory data analysis
3. Built an interactive Power BI dashboard to visualize key trends

## Key Questions Answered
- Which regions have the highest patient volume?
- What trends exist in healthcare service usage?
- How is demand distributed across different categories?

## Key Insights
- Certain regions show significantly higher patient demand
- Resource usage is uneven across regions
- Some services are consistently more utilized than others

## Files in This Project
- `data/healthcare_dataset.xlsx` → Raw dataset
- `powerbi/healthcare_dashboard.pbix` → Interactive dashboard
