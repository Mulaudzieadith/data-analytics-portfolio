# Sales Analysis End-to-End

## Objective
Analyze sales data to understand product performance, customer behavior, and regional revenue trends using Excel, SQL, and Power BI.

## Tools Used
- Excel
- SQL
- Power BI

## Dataset
The dataset includes:
- Customer information
- Product categories
- Regional sales data
- Transaction amounts

## Project Workflow
1. Cleaned and prepared the dataset in Excel
2. Used SQL to answer key business questions
3. Built a Power BI dashboard to visualize insights

## Key Questions Answered
- Which product generates the most sales?
- Which region generates the most revenue?
- Who are the top customers by spending?
- Which customers placed no orders?

## SQL Analysis
Key queries were written to:
- Identify customers without orders
- Calculate revenue by region
- Rank customers by total spending

(See `sql/analysis_queries.sql` for full queries)

## Key Insights
- Laptops generated the highest revenue
- Western Cape contributed the largest share of sales
- A small group of customers generated most revenue
- Electronics outperformed furniture overall

- `data/sales_dataset.xlsx` → Cleaned dataset
