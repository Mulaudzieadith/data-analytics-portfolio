# Sales Analysis End-to-End

## Objective
Analyze sales data to identify product performance, customer spending, and regional revenue trends using Excel, SQL, and Power BI.

## Tools Used
- Excel
- SQL
- Power BI

## Project Workflow
1. Cleaned and prepared the dataset in Excel
2. Queried the data in SQL to answer business questions
3. Built a dashboard in Power BI to visualize trends and insights

## Key Questions Answered
- Which product generated the most sales?
- Which region generated the most revenue?
- Who are the top customers by spending?
- Which customers placed no orders?

## Key Insights
- Laptops generated the highest sales revenue
- Western Cape contributed the largest share of revenue
- Electronics outperformed furniture
- A small group of customers contributed most of the revenue

## Files
- `data/` contains the cleaned dataset
- `sql/` contains SQL queries used for analysis
- `powerbi/` contains the dashboard file
- `screenshots/` contains dashboard and SQL result images
